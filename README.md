# 🌊 NOT LOST ANYMORE
### An Interactive 3D Atlantis Journey Through Learning, Growth, and Self-Discovery

## 🏛️ Overview

"Not Lost Anymore" is an immersive, AAA-quality 3D web portfolio that transforms my software engineering learning journey into an interactive underwater Atlantis city. Each lesson from my ESD (Employment Skills Development) program became a distinct district, creating a visual narrative of growth from confusion to clarity.

## 🚀 Live Demo

[portfolio-esd.netlify.app](https://portfolio-esd.netlify.app/)

---

## 🎯 The Story

Before ESD, I was moving, studying, typing, submitting. But moving where? I knew how to write code. I knew how to pass exams. But if someone had stopped me on the street and asked, "What kind of IT professional do you actually want to be?" – I would have frozen.

Not because I am stupid. But because no one had ever asked me to imagine that answer.

This portfolio is my answer. It represents everything I've learned, every perspective that shifted, and every reflection that changed me.

**From confusion to clarity. From uncertainty to confidence. Not Lost Anymore.**

## ✨ Features

### 🌊 Immersive 3D Environment
- Underwater Atlantis kingdom with bioluminescent technology
- Realistic water effects, light rays, and floating particles
- Swimming jellyfish, fish schools, and coral reefs
- AAA game-quality visuals using Three.js

### 🎮 Interactive Experience
- Click any district crystal to explore full lesson content
- Smooth camera controls with orbit and zoom
- Auto-rotate for cinematic exploration
- Fully responsive design for all devices

### 📚 Complete Content
- Every lesson from the original PDF included
- Lecturer names, dates, activities, and assignments
- Personal reflections and key concepts highlighted
- No content summarized, shortened, or skipped
